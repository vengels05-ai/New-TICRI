# Arizona Constitution - Download Guide

**Official Source**: https://azleg.gov/constitution/
**Last Updated**: January 12, 2026

## Structure

The Arizona Constitution has:
- 1 Preamble
- 30 Articles (numbered 1-30, with some gaps)

## Manual Download Instructions

1. Visit each URL above
2. Copy the full text from each page
3. Combine into one file: `AZ-raw.txt`
4. Format: 

```
ARIZONA STATE CONSTITUTION

PREAMBLE
[preamble text]

ARTICLE 1 - STATE BOUNDARIES
[article 1 text]

ARTICLE 2 - DECLARATION OF RIGHTS
[article 2 text]

... etc
```

## Notes

- Arizona's constitution is well-organized
- Each article is on a separate page
- Clean HTML, easy to copy
