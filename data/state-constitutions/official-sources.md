# Official State Constitution Sources

This document contains the official government website URLs for each state's constitution.

## State Constitution URLs

### Alabama
- **Official Source**: Alabama Legislature
- **URL**: http://alisondb.legislature.state.al.us/alison/CodeOfAlabama/Constitution/1901/CA-245731.htm

### Alaska
- **Official Source**: Alaska State Legislature
- **URL**: http://www.legis.state.ak.us/basis/folioproxy.asp?url=http://wwwjnu01.legis.state.ak.us/cgi-bin/folioisa.dll/acontxt/query=*/doc/{t750}

### Arizona
- **Official Source**: Arizona State Legislature
- **URL**: https://azleg.gov/const/

### Arkansas
- **Official Source**: Arkansas General Assembly
- **URL**: https://www.arkleg.state.ar.us/Bills/Document?type=pdf&source=static&file=ARConstitution1874.pdf

### California
- **Official Source**: California Legislature
- **URL**: https://leginfo.legislature.ca.gov/faces/codes_displayexpandedbranch.xhtml?lawCode=CONS
- **Full Text URL Pattern**: https://leginfo.legislature.ca.gov/faces/codes_displayText.xhtml?lawCode=CONS&division=&title=&part=&chapter=&article=[ARTICLE]

### Colorado
- **Official Source**: Colorado General Assembly
- **URL**: https://leg.colorado.gov/colorado-constitution

### Connecticut
- **Official Source**: Connecticut General Assembly
- **URL**: https://www.cga.ct.gov/current/pub/chap_000.htm

### Delaware
- **Official Source**: Delaware General Assembly
- **URL**: https://delcode.delaware.gov/constitution/

### Florida
- **Official Source**: Florida Legislature
- **URL**: http://www.leg.state.fl.us/Statutes/index.cfm?Mode=Constitution&Submenu=3&Tab=statutes

### Georgia
- **Official Source**: Georgia General Assembly
- **URL**: https://www.legis.ga.gov/document/constitution/

### Hawaii
- **Official Source**: Hawaii State Legislature
- **URL**: https://www.capitol.hawaii.gov/hrscurrent/Vol01_Ch0001-0042F/00-Hawaii_Constitution/

### Idaho
- **Official Source**: Idaho Legislature
- **URL**: https://legislature.idaho.gov/statutesrules/idconst/

### Illinois
- **Official Source**: Illinois General Assembly
- **URL**: https://www.ilga.gov/commission/lrb/con
